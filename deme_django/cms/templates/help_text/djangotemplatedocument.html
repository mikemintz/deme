A Django template document can be used to affect the website's entire layout.

By default, Deme uses Bootstrap to layout the page. For example, to have content in two columns, enter "<div class='row'><div class='col-sm-6'>First Column</div><div class='col-sm-6'>Second Column</div></div>". Learn more from the Bootstrap docs.

For more details on how to use Django template documents, see the Deme documentation.
