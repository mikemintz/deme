Custom URLs let you create a path to an item.

For example, to have http://www.yoursite.com/about/ point to an HTML document named "About Page", set:

  1. Item name to "About Page URL"
  2. Viewer to "htmldocument"
  3. Action to "show"
  4. Aliased item to the About Page item
  5. Parent URL to your base Site item
  6. Path to "about"

To set up http://www.yoursite.com/about/team, you'd set Parent URL to the Custom URL for the about page, to the "About Page URL" item.
